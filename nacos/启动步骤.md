从 http://qiniu.lancaiwu.com/nacos-server.jar  下载  nacos-server.jar
nacos-server.jar 放到 target文件夹里面去(若没有，则新建在当前目录 \nacos\target)

1、 cd  nacos/bin

###  linux 启动

2、 sh startup.sh -m standalone

###  windows 启动

2、 .\startup.sh -m standalone


++++++++++

###  linux 停止步骤

3、 sh shutdown.sh

###  windows 停止步骤

3、 .\shutdown.cmd
